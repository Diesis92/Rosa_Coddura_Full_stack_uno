Tabella Contenuti

TAG Attributi Descrizione
p Style, id, class Definisce un paragrafo
a href Collegamento ipertestuale
img src, alt, width, height Includere un'immagine
video src, controls Includere un video
audio src, controls Includere un file audio
blockquote cite Definisce una citazione lunga
ul None Lista non ordinata
ol None Lista ordinata
li None Elemento di lista

Tabella Layout

TAG Attributi Descrizione
div id, class, style Contenitore generico per elementi
span id, class, style Contenitore in linea per porzioni di testo
section None Sezione di contenuto tematico
article None Contenuto autonomo (come un articolo)
header None Intestazione di una pagina o sezione
footer None Piè di pagina di una pagina o sezione
nav None Contenitore per link di navigazione
aside None Contenuto marginale, spesso in una barra laterale

Tabella Contenitori

TAG Attributi Descrizione
html lang, dir Elemento radice di un documento HTML
head None Contiene metadati e link a risorse esterne
body None Contenuto visibile della pagina
title None Titolo del documento (visibile nella tab del browser)
meta charset, name, content Metadati del documento (descrizione, autore, charset)
link rel, href Collega risorse esterne (es. fogli di stile CSS)
script src, type Aggiunge uno script JavaScript alla pagina
style None Contiene gli stili CSS direttamente nel documento
noscript None Contenuto alternativo se il browser non supporta JavaScript
