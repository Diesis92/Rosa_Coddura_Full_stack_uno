Tag HTML5 Divisi per Categoria

Contenuti

Tag Descrizione
<p> Paragrafo di testo
<h1>...<h6> Titoli (da h1 a h6)
<a> Collegamento ipertestuale
<img> Immagine
<audio> Audio (file audio)
<video> Video (file video)
<blockquote> Citazione lunga
<q> Citazione breve in linea
<strong> Testo importante (grassetto)
<em> Testo enfatizzato (corsivo)
<code> Segmento di codice
<mark> Testo evidenziato
<br> Interruzione di linea (a capo)
<ul> Lista non ordinata (con punti elenco)
<ol> Lista ordinata (con numeri)
<li> Elemento di lista
<dl> Lista di definizione
<dt> Termine di una lista di definizione
<dd> Descrizione di un termine in una lista di definizione

Layout

Tag Descrizione
<div> Contenitore generico per raggruppare altri elementi
<span> Contenitore in linea per gruppi di testo o elementi
<section> Sezione di contenuti tematici
<article> Contenuto autonomo (es. articolo di blog)
<aside> Contenuto marginale (es. barra laterale)
<header> Intestazione di un documento o di una sezione
<footer> Piè di pagina di un documento o di una sezione
<nav> Area di navigazione per link principali
<main> Contenuto principale del documento
<figure> Contenitore per media (es. immagini, grafici)
<figcaption> Didascalia di un elemento <figure>
<details> Contenuto espandibile/collassabile (con <summary>)
<summary> Sintesi di un contenuto <details>

Contenitori

Tag Descrizione
<html> Elemento radice di un documento HTML
<head> Contiene metadati e link a risorse esterne
<body> Contenuto visibile della pagina
<title> Titolo del documento (visibile nella tab del browser)
<meta> Metadati per la pagina (descrizione, charset, etc.)
<link> Collega risorse esterne (come CSS)
<script> Aggiunge script JavaScript alla pagina
<style> Contiene stili CSS direttamente nel documento
<base> Definisce un URL di base per tutti i collegamenti relativi
<noscript> Contenuto alternativo se il browser non supporta JavaScript
