1️⃣ Validazione quantità con clean_quantita()

Attualmente controlli la quantità nel clean() del form. L'esercizio ti chiede di creare una validazione dedicata.

In forms.py
class AcquistoForm(forms.Form):
    prodotto = forms.ModelChoiceField(
        queryset=Prodotto.objects.all()
    )

    quantita = forms.IntegerField(min_value=1)

    codice_sconto = forms.CharField(
        required=False
    )

    def clean_quantita(self):
        quantita = self.cleaned_data["quantita"]

        if quantita <= 0:
            raise forms.ValidationError(
                "La quantità deve essere maggiore di zero"
            )

        return quantita
Perché?

Django esegue automaticamente:

clean_nomecampo()

per validare un singolo campo.

Questa validazione riguarda solo quantita, quindi è più corretto usare clean_quantita() invece di clean().

def clean(self):

    cleaned_data = super().clean()
